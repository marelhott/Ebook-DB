# Frontend

Minimal Next.js-style scaffold for the book platform.

## Included

- app shell
- home page
- library page
- author page
- book detail page
- reader shell
- live EPUB reader scaffold via `epub.js`
- search page
- import wizard
- design tokens
- global styling
- API fallback layer

## Notes

- Dependencies have been installed locally for validation
- `npm run build` currently passes
- The frontend can read from backend API when available and falls back to local mock data for development
- `frontend/public/samples/alice.epub` is included as a local reader sample
